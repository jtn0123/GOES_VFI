Support
=======

Getting Help
------------

If you need help with GOES_VFI, there are several resources available:

GitHub Issues
~~~~~~~~~~~~~

For bug reports and feature requests, please use the GitHub issue tracker:

- **Bug Reports**: https://github.com/GOES_VFI/GOES_VFI/issues
- **Feature Requests**: https://github.com/GOES_VFI/GOES_VFI/issues

When reporting a bug, please include:

1. Your operating system and Python version
2. GOES_VFI version
3. Complete error messages and stack traces
4. Steps to reproduce the issue
5. Any relevant configuration or code snippets

GitHub Discussions
~~~~~~~~~~~~~~~~~~

For general questions and discussions:

- **Discussions**: https://github.com/GOES_VFI/GOES_VFI/discussions

Frequently Asked Questions
--------------------------

**Q: What versions of Python are supported?**
   A: GOES_VFI requires Python 3.13 or higher.

**Q: How do I report a security vulnerability?**
   A: Please report security vulnerabilities privately to the maintainers.

**Q: Can I use GOES_VFI for commercial purposes?**
   A: Yes, GOES_VFI is licensed under the MIT License which allows commercial use.

**Q: Where can I find example scripts?**
   A: Example scripts are available in the `examples/` directory of the repository.

**Q: How can I improve processing performance?**
   A: See the performance tuning section in the user guide for optimization tips.
