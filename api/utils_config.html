popitem()

Remove and return a (key, value) pair as a 2-tuple.

Pairs are returned in LIFO (last-in, first-out) order. Raises KeyError if the dict is empty.

setdefault(key, default=None, /)

Return the value for key if key is in the dictionary, else default.

update([E, ]**F) → None.  Update D from mapping/iterable E and F.

If E is present and has a .keys() method, then does: for k in E.keys(): D[k] = E[k] If E is present and lacks a .keys() method, then does: for k, v in E: D[k] = v In either case, this is followed by: for k in F: D[k] = F[k]

goesvfi.utils.config.find_rife_executable(model_key: str) → Path[source]

Locate the RIFE CLI executable. Searches in order:

  1. System PATH for ‘rife-ncnn-vulkan’ (or .exe)

  1. Project ‘goesvfi/bin/’ directory for ‘rife-cli’ (or .exe)

  2. Project ‘goesvfi/models/<model_key>/’ directory for ‘rife-cli’ (or .exe).
