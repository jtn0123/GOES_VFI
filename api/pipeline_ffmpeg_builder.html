Ffmpeg Builder

ffmpeg_builder.py.

Provides the FFmpegCommandBuilder class for constructing FFmpeg command-line arguments in a flexible and modular way, supporting various encoding options and hardware/software backends for the GOES_VFI pipeline.

class goesvfi.pipeline.ffmpeg_builder.FFmpegCommandBuilder[source]

Bases: object

Builder for FFmpeg command-line arguments for video encoding.

This class provides a fluent interface for specifying input/output files, encoder options, quality/bitrate settings, pixel format, and two-pass encoding parameters. It generates a command list suitable for subprocess execution.

Typical usage:

builder = FFmpegCommandBuilder() cmd = (

builder.set_input(input_path) .set_output(output_path) .set_encoder(“Software x265”) .set_crf(20) .set_pix_fmt(“yuv420p10le”) .build()

)
