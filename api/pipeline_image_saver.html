Image Saver

image_saver.py.

Provides the ImageSaver class, an ImageProcessor implementation for saving images to disk using Pillow.

class goesvfi.pipeline.image_saver.ImageSaver[source]

Bases: ImageProcessor

ImageProcessor implementation for saving images to disk using Pillow.

This class saves ImageData objects to disk as image files. It does not implement loading, processing, or cropping.

save(image_data: ImageData, destination_path: str) → None[source]

Save image data to the specified destination path using Pillow.

Parameters:

  • image_data (ImageData) – The image data to save.

  • destination_path (str) – The path where the image will be saved.

Raises:

  • IOError – If there’s an error writing the file.

  • ValueError – If the image data cannot be saved.
