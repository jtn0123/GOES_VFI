goesvfi.pipeline.tiler.tile_image(img: numpy.typing.NDArray.numpy.float32, tile_size: int = 2048, overlap: int = 32) → list[tuple[int, int, numpy.typing.NDArray.numpy.float32]][source]

Split H×W×3 image into overlapping RGB float32 tiles.

Parameters:

  • img – Full image to tile in H×W×3 format.

  • tile_size – Size of the square tile. The actual tile may be smaller at the image edges.

  • overlap – Amount of overlap in pixels between tiles. Set to 0 for no overlap.
