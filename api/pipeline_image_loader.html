Image Loader

image_loader.py.

Provides the ImageLoader class, an ImageProcessor implementation for loading images from disk using Pillow and converting them to ImageData objects for the GOES_VFI pipeline.

class goesvfi.pipeline.image_loader.ImageLoader(optimize_memory: bool = True, max_image_size_mb: int | None = None)[source]

Bases: ImageProcessor

ImageProcessor implementation for loading images from disk using Pillow.

This class loads image files into ImageData objects, extracting pixel data and basic metadata. It does not implement processing, cropping, or saving.
