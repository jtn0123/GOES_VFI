Rife Analyzer

RIFE CLI Analyzer - Utility to detect RIFE CLI capabilities.

This module provides functionality to analyze the capabilities of the RIFE CLI executable, helping to bridge the gap between what the application expects and what the executable actually supports.

Detects and reports on the capabilities of a RIFE CLI executable.

This class runs the RIFE executable with various flags to determine which options are supported, allowing the application to adapt its behavior accordingly.
