Log

Colourised logging helper using colorlog when available.

goesvfi.utils.log.get_logger(name: str | None = None) → Logger[source]

Return a module-specific logger with optional color formatting.

goesvfi.utils.log.set_global_log_level(level: int) → None[source]

Configure the root logger with a single handler and set its level. This relies on propagation for named loggers.

goesvfi.utils.log.set_level(debug_mode: bool) → None[source]

Set the global logging level based on debug mode.
