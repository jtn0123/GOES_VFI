Image Cropper

image_cropper.py.

Provides the ImageCropper class, an ImageProcessor implementation for cropping images to a specified rectangular area in the GOES_VFI pipeline.

class goesvfi.pipeline.image_cropper.ImageCropper[source]

Bases: ImageProcessor

ImageProcessor implementation for cropping images to a specified area.

This class provides a crop method that extracts a rectangular region from an image.
