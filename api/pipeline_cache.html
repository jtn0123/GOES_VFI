Cache

Utility helpers for caching interpolated frames on disk.

Frames are saved as .npy files using a SHA-256 hash derived from the two input frame paths, the RIFE model identifier and the number of interpolated frames. When the same inputs are processed again the cached .npy files are loaded instead of recomputing the interpolation.

goesvfi.pipeline.cache.load_cached(path1: Path, path2: Path, model_id: str, num_intermediate_frames: int) → list[numpy.typing.NDArray.typing.Any] | None[source]

Load cached intermediate frames if all .npy files are present.
